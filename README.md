# 🚀 Fikir Atölyesi (Thinking Workshop)
**NAIM Framework: 170kg Boss Level Milestone Reached! 👑**
Fikir Atölyesi, aklınızdaki ham fikirleri saniyeler içinde işlenebilir projelere, dijital araçlara ve somut yol haritalarına dönüştüren premium bir mobil asistandır. NAIM (Naim Agentic Iterative Mobile) prensipleriyle geliştirilerek 14 iterasyonda "Boss Level" seviyesine ulaştı.
---
## ✨ Temel Özellikler
### 🧠 AI Destekli Dönüşüm (Boss Level)
Yazdığınız her fikir, AI algoritması tarafından analiz edilir ve o fikri hayata geçirmek için **3 somut aksiyon adımı** (Next Steps) içeren bir "AI Yol Haritası" oluşturulur.
### 💾 Kesintisiz Kalıcılık (Persistence)
`AsyncStorage` entegrasyonu sayesinde tüm fikirleriniz, başlıklarınız, AI analizleriniz ve tema tercihleriniz telefonunuzun hafızasında güvenle saklanır. Uygulama kapansa dahi verileriniz asla kaybolmaz.
### 🌓 Dinamik Tema Sistemi
Tek bir dokunuşla Koyu (Dark) ve Aydınlık (Light) modlar arasında geçiş yapın. Tüm arayüz, glassmorphism estetiğini koruyarak akıcı bir şekilde renk değiştirir.
### 🎬 Akışkan Animasyonlar & Haptic Geri Bildirim
`LayoutAnimation` ve `Animated` API kullanılarak hazırlanan mikro-animasyonlar sayesinde listeden eleman silme, ekleme ve düzenleme işlemleri "premium" bir hissiyat sunar.
### 📝 Gelişmiş Editör
Kaydettiğiniz fikirleri dilediğiniz zaman başlık, içerik veya AI adımları bazında güncelleyebilir, fikirlerinizi evrimleştirebilirsiniz.
---
## 🛠️ Teknoloji Yığını
- **Framework:** React Native / Expo (SDK 52)
- **State Management:** React Context API (IdeasProvider)
- **Veri Saklama:** @react-native-async-storage/async-storage
- **İkon Seti:** Lucide React Native
- **Navigasyon:** React Navigation (Stack)
- **Animasyon:** LayoutAnimation & Animated API
- **UI Tasarımı:** Custom CSS / Glassmorphism Design System
---
## 🚀 Nasıl Çalıştırılır?
Projeyi yerel ortamınızda çalıştırmak için:
1. **Bağımlılıkları Yükleyin:**
   ```bash
   npm install
   ```
2. **Uygulamayı Başlatın:**
   ```bash
   npx expo start
   ```
3. **Expo Go ile İzleyin:**
   Terminalde çıkan QR kodu telefonunuzdaki Expo Go uygulamasıyla taratın.
